# doctrine2-spatial

This is a fork of https://github.com/djlambert/doctrine2-spatial with additional functions for PostGIS:

* ST_Buffer
* ST_Difference
* ST_EndPoint
* ST_GeometryN
* ST_Line_Locate_Point
* ST_Line_Substring
* ST_Split
* ST_Translate
* ST_Union
* ST_X
* ST_Y

refer to the parent repository for all other instructions
